The same pass that generates the card title also returns a suggested follow-on skill. The pill updates as the user types (debounced). A small orange dot marks it as AI-suggested — clicking the pill and choosing the same option removes the dot (user-confirmed).
Skill subset
Only the four inline skills appear here: Workshop, Interview, Draft spec, Implement. Subagent skills (review, audit) and post-implementation skills (PR, Fix CI) are not relevant at creation time.
No selection default
If the user types slowly or the AI hasn't returned a suggestion yet, the pill reads "After creation…" in muted text. The card is still createable — the follow-on is optional.
