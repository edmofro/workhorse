PR controls in the artifacts sidebar

Two placement options for surfacing pull request status within the artifacts sidebar. Option A places PR info at the very top as a distinct section. Option B embeds it within the Code changes section.

Option A — Top of sidebar

PR section above all file sections

A compact row at the top of the sidebar showing PR status, number, and a link out. Distinct from the file sections below.
