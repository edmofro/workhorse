PR in sidebar — interactive options

The PR section at the top of the artifacts sidebar replaces the properties bar chip. These mockups show where branch details, auto-fix toggle, and actions live.

Option 1 — Popover

Clicking the PR row opens a popover to the left of the sidebar

The sidebar row is the trigger. All detail — PR card, auto-fix toggle, branch diagnostics — lives in a floating popover. Same content as the old chip popover, just anchored to the sidebar.
