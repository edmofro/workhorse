Editing state

Row 3: "In review"

Shows a newly added custom status being edited. The label is in an inline text input with accent focus ring. The colour category picker dropdown is open, showing the four fixed options.

Role tags

Done and Cancelled

The "Complete" and "Cancelled" rows show small role tags. These statuses have special behaviour (green dot for done, collapsed column for cancelled) and cannot be deleted — their delete buttons are disabled.

Drag handle

Reordering

The grip icon on each row allows drag-to-reorder. The cancelled status has no drag handle — it is always last. The done status can be reordered but stays after all regular statuses.

Colour categories

Fixed palette

Four options: Grey (not started / backlog), Amber (in progress), Green (done), Muted (cancelled / inactive). These map directly to the design system's status dot colours. Not user-extensible.

Deleting a status

Card reassignment

When deleting a status that has cards, a confirmation dialog asks the user to choose a replacement status before proceeding. Statuses with the done or cancelled role cannot be deleted.
